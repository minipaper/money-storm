# money-storm aka 돈벼락

하이킨 아시를 이용한 봇 테스트

```
종가 = 1/4(시가+종가+저가+종가) :: 평균값 계산
시가 = 1/2(이전 캔들의 시가 + 이전 캔들의 종가) :: 이전 캔들의 중간 값
고가 = 최대값(고가/시가/종가)
저가 = 최저값(저가/시가/종가)
```

## 실행방법

`.env.template` 파일을 `.env`로 복사하고 발급받은 API KEY 입력 `.env`파일은 ignore 처리되어 버전관리 안함 하면안됨

`npm install` - 최초 필요한 라이브러리 설치

`npm start` - app 시작
